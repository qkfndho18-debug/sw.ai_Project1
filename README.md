# 🌱 StudyBloom

## 학생 맞춤형 자기주도 학습 관리 프로그램

### 프로젝트 소개

StudyBloom은 학생의 학습 성향을 분석하고,
그 결과를 바탕으로 개인 맞춤형 공부 계획을 추천하는
객체지향(Object-Oriented Programming) 기반 학습 관리 프로그램입니다.

학생 스스로 자신의 공부 습관을 이해하고,
목표를 세우며,
공부 기록을 관리하고,
피드백을 받아 자기주도학습 능력을 기를 수 있도록 설계되었습니다.

---

## 프로젝트 목적

기존 학교나 학원에서는 학생 개인의 학습 성향과 공부 습관을
충분히 고려하기 어려운 경우가 많습니다.

StudyBloom은 학생마다 다른 학습 성향을 분석하여
자신에게 맞는 공부 방법을 추천하고,
학생 스스로 학습 계획을 세울 수 있도록 돕는 것을 목표로 합니다.

### 👤 학생 관리

- 학생 정보 등록
- 레벨 관리
- 경험치 시스템

---

- 학습 성향 테스트

을 제공합니다.

- 공부 과목 저장
- 공부 시간 저장
- 기록 조회

## 객체지향 설계

### Student

학생 정보 관리

### Analyzer

학습 성향 분석

### StudyPlan

맞춤 공부 계획 생성

### RecordManager

공부 기록 저장 및 조회

### Feedback

맞춤 피드백 제공

### LevelSystem

레벨 및 경험치 관리

### Quiz

오늘의 퀴즈
